# 🚀 MechMind OS su GitHub Codespaces

Zero installazioni locali! Tutto nel cloud. Gratis 120 ore/mese.

## 🎯 Avvio in 3 click

1. Vai su: `https://github.com/codespaces/new?repo=tnx9pv2rgt-bot/mechmind-os`
2. Click **"Create codespace on main"**
3. Aspetta 3-4 minuti

## 🌐 URL Pubblici

| Servizio | URL |
|----------|-----|
| 🎨 **Frontend** | `https://TUO-CODESPACE-3001.app.github.dev` |
| ⚡ **Backend** | `https://TUO-CODESPACE-3000.app.github.dev` |
| 🗄️ **Adminer** | `https://TUO-CODESPACE-8080.app.github.dev` |

## 🆓 Quota Gratuita

- **120 ore/mese** CPU
- **15GB storage**
- 2 core, 8GB RAM

## 📋 Comandi

```bash
docker-compose logs -f   # Log
docker-compose down      # Ferma
docker-compose up -d     # Riavvia
```
