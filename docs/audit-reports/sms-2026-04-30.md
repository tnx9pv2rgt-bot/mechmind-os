## Root Cause Analysis (BLOCCANTI)

### BUG-1: PII SMS Body in Plaintext
```typescript
// PRIMA (VULN):
await this.prisma.smsMessage.create({ data: { body } });  // plaintext

// DOPO (SAFE):
await this.prisma.smsMessage.create({ data: { body: this.encryption.encrypt(body) } });
// + decrypt on read in getMessages() and receiveInbound() idempotency path
```
**Impatto:** GDPR Art.32 violation — customer SMS content stored unencrypted.
**Traceability:** GDPR Art.32, OWASP A02:2021, CLAUDE.md PII Rule
