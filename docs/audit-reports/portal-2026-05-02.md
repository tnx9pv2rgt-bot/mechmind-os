## Root Cause Analysis — BLOCCANTI

### BUG-1: Assertion Density Crisis (F-001)
**Issue:** Average 1.48 assertions/test; standard requires ≥2.0

**Code Sample (BEFORE):**
```typescript
it('getDashboard returns upcomingBooking', async () => {
  const result = await service.getDashboard(CUSTOMER_ID, TENANT_ID);
  expect(result).toBeDefined();  // Only 1 assertion
});
```

**Fix (AFTER):**
```typescript
it('getDashboard returns upcomingBooking with vehicle details', async () => {
  const result = await service.getDashboard(CUSTOMER_ID, TENANT_ID);
  
  // Assertion 1: Structure
  expect(result.data).toHaveProperty('upcomingBooking');
  
  // Assertion 2: Behavior - verify tenantId filtered correctly
  expect(mockPrisma.booking.findFirst).toHaveBeenCalledWith(
    expect.objectContaining({ where: expect.objectContaining({ tenantId: TENANT_ID }) })
  );
  
  // Assertion 3: Encryption verified
  expect(result.data.customer.phone).not.toContain('enc:');
});
```

**Impact:** Missing assertions mask false-positive tests (code passes but doesn't actually verify behavior)

**Traceability:** Coverage standard 90/90, test-quality-gates.md STEP 5

**DORA Rework:** YES — architectural debt

### BUG-2: Mock State Pollution (F-002)
**Issue:** 251 lines use `mockResolvedValue` without `Once` suffix outside beforeEach

**Code Sample (BEFORE):**
```typescript
describe('getVehicles', () => {
  let service: PortalService;

  beforeEach(async () => {
    // ... setup
  });

  it('test 1: returns vehicles', async () => {
    prisma.vehicle.findMany.mockResolvedValue([{ id: 'v1' }]); // ❌ NO ONCE
    await service.getVehicles(...);
  });

  it('test 2: filters by tenant', async () => {
    // This test INHERITS mockResolvedValue from test 1!
    await service.getVehicles(CUSTOMER_ID, TENANT_DIFFERENT);
  });
});
```

**Fix (AFTER):**
```typescript
describe('getVehicles', () => {
  beforeEach(async () => {
    jest.clearAllMocks();
    // Set default for all tests
    prisma.vehicle.findMany.mockResolvedValue([{ id: 'v1', tenantId: TENANT_ID }]);
  });

  it('test 1: returns vehicles', async () => {
    // Use default OR override with Once
    prisma.vehicle.findMany.mockResolvedValueOnce([{ id: 'v1' }]);
    await service.getVehicles(...);
  });

  it('test 2: filters by tenant', async () => {
    // Clean slate — uses beforeEach default or own Override
    await service.getVehicles(CUSTOMER_ID, TENANT_DIFFERENT);
  });
});
```

**Impact:** Test pollution causes intermittent failures, false confidence in passing test suites

**Traceability:** Test quality gates STEP 6, jest-best-practices

**DORA Rework:** YES — stability risk

**Example from service.ts (line 52-58 uncovered):**

```typescript
const customer = await this.prisma.customer.findFirst({

if (!customer) {  // ← UNCOVERED: error path
```

**Why uncovered:** All tests set up `mockCustomer` successfully; none test null case

**Fix (ADD TEST):**

```typescript
it('getDashboard throws NotFoundException when customer not found', async () => {

  );
});
```

**Impact:** Untested error paths hide bugs; production failures for edge cases

**Traceability:** Coverage standard 90/90, OWASP A10:2025 exception handling

**Fixed lines:** 1688, 1689, 1710, 1711, 1732, 1753, 1771, 1792, 1793, 2118, 2119, 2140, 2141, 2162, 2163, 2181, 2199, 2217

```typescript
// BEFORE
expect(result.data[0].alerts).toContain('REVISION_EXPIRED');
```
